Projeto - Yu-Gi-Oh!

Descrição

Aplicação frontend que simula uma loja de cartas Yu-Gi-Oh!, baseada em layout do Figma.

O usuário pode:

Pesquisar cartas pelo nome na barra de busca (header fixo).

Filtrar cartas por atributos e tipos na sidebar.

Adicionar cartas ao carrinho de compras.

Visualizar subtotal, custo de frete e total a pagar.

Finalizar a compra com feedback em modais estilizados.

O objetivo é reproduzir fielmente o layout do Figma, mantendo uma experiência fluida e responsiva.

Tecnologias Utilizadas

React.js (Vite) – Framework principal

TailwindCSS – Estilização responsiva e customizada

React Router DOM – Navegação entre páginas

Context API – Gerenciamento de estado global (carrinho)

Como Rodar o Projeto

1. Clonar o repositório
   git clone https://github.com/paulovsc1/teste-fpr.git
   cd teste-fpr

2. Instalar dependências
   npm install

3. Executar
   npm run dev

Acesse no navegador: http://localhost:5173
